# Kirana Store Home Delivery

A simple grocery storefront with:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: MySQL

## Run locally
1. Install dependencies: npm install
2. Copy .env.example to .env and set your MySQL credentials.
3. Create the MySQL database using database.sql.
4. Start the app: node server.js

The app will be available at http://localhost:3000

## Automatic deployment
This project includes a Render deployment config and a GitHub Actions workflow.

### Render
1. Create a free Render account.
2. Create a new Web Service and connect this repository.
3. Render will use the included render.yaml file automatically.
4. Your public link will be generated by Render after deployment.

### GitHub Actions
1. Add a secret named RENDER_DEPLOY_HOOK_URL in your GitHub repository settings.
2. Create a deploy hook in Render and paste the URL into the secret.
3. Push to the main branch to trigger deployment.

### Quick public-hosting steps
1. Push this folder to GitHub.
2. Open Render and create a new Web Service.
3. Connect the GitHub repo.
4. Deploy.
5. Copy the public URL from Render.
